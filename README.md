# RLMM
Robust Linear Mixed Models
RLMM is a R package that fits longitudinal data using robust loss functions such as Huber and Tukey. 

# References
I.M. Danilevicz, P. Bondon, V. A. Reisen, F. Sarquis. A longitudinal study of the influence of air pollutants on children: A robust multivariate approach. Journal of Applied Statistics, 2023, ⟨10.1080/02664763.2023.2272228⟩.
